- Quirky "git diff" and "git show" behavior, where Color Escape sequences were not properly interpreted, leaving lots of "ESC[<N>m" gibberish in the output. Found out that for this latest Git version, it is handy to set the "LESS" environment variable to simply "-R", as in:

   export LESS="-R"

   -> This provided true Color Diff output in a terminal shell. Yay! :-D
